## ðŸš€ Getting Started

Follow these steps to run the project locally:

### Prerequisites

* Node.js >= 14.x
* npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Rexpass/SaaS-Landing-Page-React.js.git

```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
```
